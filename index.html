Hi javascript developer! Welcome to Slatebox.js

Slatebox.js is the javascript library behind Slatebox.com's mind-mapping and concept drawing. This source even includes the signalr hooks to wire up real-time collaboration.

2 Minute Startup

To make it really easy to use, simply include a reference to the 71kb gzipped and minified Slatebox.js library (which includes the full raphael and signalr libraries -- two Slatebox dependencies) as well as jquery (only required for SignalR collaboration).

<script src="http://ajax.aspnetcdn.com/ajax/jquery/jquery-1.8.0.min.js" type="text/javascript"></script>
<script type="text/javascript" src="http://static.slatebox.com/dev/js/0.5.0/slatebox.min.js"></script>
